# Update Project Memory

When a meaningful architecture, protocol, or workflow decision changes:

1. Update `plan.md`.
2. Update `.codex/project-memory.md`.
3. Add a concise entry to `.codex/change-log.md`.
4. If a blocker was resolved, update `.codex/issues-and-resolutions.md`.
5. If the decision affects protocol or boundaries, update the matching file in `docs/`.
